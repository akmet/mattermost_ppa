To use this PPA: 1. Add GPG key curl -s --compressed https://akmet.github.io/mattermost_ppa/KEY.gpg | sudo apt-key add - 2. Download repository information sudo curl -s --compressed -o /etc/apt/sources.list.d/mattermost_ppa.list "https://akmet.github.io/mattermost_ppa/mattermost_ppa.list" 3. Install mattermost-desktop sudo apt update && sudo apt install mattermost-desktop
